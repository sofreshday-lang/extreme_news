# Extreme News Final Dashboard

본 프로젝트는 네이버 뉴스 API를 활용하여 관심 키워드를 실시간으로 모니터링하고, 중복 데이터를 제거하여 최적의 인사이트를 제공하는 풀스택 웹 애플리케이션입니다.

## 🚀 주요 기능
1. **방안 A 독립 검색 시스템**:
   - **카테고리 선택**: 사전 정의된 11개 키워드를 즉시 검색.
   - **직접 입력**: 사용자가 원하는 검색어를 입력하여 분석.
   - 두 영역은 각각 독립적으로 작동하거나, 하단 버튼을 통해 통합 분석할 수 있습니다.
2. **AND/OR 검색 모드**:
   - **OR 모드**: 각 키워드별로 개별 섹션에 뉴스 출력.
   - **AND 모드**: 모든 선택 키워드가 포함된 교집합 뉴스만 정밀 검색.
3. **자동 키워드 보강**:
   - 검색 정확도를 높이기 위해 검색어 뒤에 `"식품" "약품" "건강"` 수식어를 자동 조합.
4. **스마트 기간 필터링**:
   - 최근 24시간~30일 퀵 선택 및 달력(Calendar)을 통한 상세 범위 지정.
5. **고급 중복 제거**:
   - URL 매칭 및 제목 유사도(80%) 분석을 통해 최신 기사 1건만 노출.

## 🛠️ 설치 및 실행 (로컬)
1. `.env` 파일에 네이버 API 키를 설정합니다. (절대 외부 노출 금지)
   - `NAVER_CLIENT_ID`
   - `NAVER_CLIENT_SECRET`
2. 명령어를 실행합니다:
   ```bash
   python local_server.py
   ```
3. 브라우저에서 `http://localhost:8000` 접속.

## ☁️ Vercel 배포 방법
1. GitHub 저장소에 코드를 Push합니다.
2. Vercel 프로젝트 설정의 **Environment Variables**에 위 API 키 2개를 추가합니다.
3. 배포가 완료되면 사내 공유 URL을 획득할 수 있습니다.
